# 🛍️ Customer Segmentation using K-Means Clustering (Task 2)

This project uses the K-Means clustering algorithm to group customers of a retail store based on their purchase behavior.

---

## 🚀 Features

- Groups customers into different clusters  
- Helps understand customer behavior  
- Visualizes clusters using graphs  
- Simple and effective unsupervised learning model  

---

## 🛠️ Technologies Used

- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib  

---

## 📂 Project Structure

customer-segmentation/
│── dataset.csv  
│── model.py  
│── README.md  

---

## ▶️ How to Run

1. Download or clone this repository  
2. Install required libraries:


pip install numpy pandas scikit-learn matplotlib


3. Run the Python file:


python model.py


---

## 🎯 Objective

To group customers into different segments based on their purchasing patterns using K-Means clustering.

---

## 📊 Model Used

- K-Means Clustering  

---

## 📈 Output

- Customers are divided into clusters  
- Visualization of clusters using scatter plot
